Installation
------------

.. parsed-literal::

    tar xvzf qdpy-|version|.tar.gz
    cd qdpy-|version|
    python setup.py build
    sudo python setup.py install

The qdpy source code is `hosted on GitHub <https://github.com/jerluc/qdpy>`_.
